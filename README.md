# 🌍 JiKiGo
> **"여행을 사랑하는 우리가 환경을 보호할 수 있는 방법은 없을까요?"**  

우리가 **지키는 환경이 곧 우리가 누릴 여행의 미래**가 된다는 메시지를 담아, 지속 가능한 여행 문화를 만들어가고자 합니다.
